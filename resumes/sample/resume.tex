% ============================================================
% TWO-COLUMN "RAIL" LAYOUT BUILDING BLOCKS (page-breakable)
% ============================================================
% Left column (~28% width) carries per-entry metadata: job title /
% company / dates / location, or degree / school / date. Right column
% carries the actual content: bullets, or a description paragraph.
% A full-width rule sits above each entry.
%
% Implementation notes (the hard-won parts):
% 1. A true minipage/tabular approach can't break across pages, which
%    breaks long sections (e.g. Employment History spanning multiple
%    pages). So the right-column content is laid out as a normal
%    indented block (flows/breaks naturally), and the left-column
%    metadata is placed in the margin via \makebox.
% 2. A naive \makebox[0pt][r]{\parbox[t]{...}{multi-line metadata}}
%    looks fine for short (1-2 line) labels, but for a 4-line label
%    (Title/Company/Dates/Location) the box's full DEPTH gets counted
%    by TeX's interline spacing, pushing the following line (the first
%    bullet) down by roughly the box's height. Fix: force the box's
%    depth to 0pt via \raisebox{0pt}[\ht][0pt]{...} while preserving
%    its real height, so alignment is correct AND the page-builder
%    still reserves the correct vertical space.
% 3. Because the metadata box's height is now "hidden" inside a
%    same-line construct, ordinary LaTeX orphan/widow avoidance no
%    longer naturally keeps a job's header attached to its first
%    bullet across a page break. \needspace reserves room for the
%    rule + a few lines before starting a new entry, so an entry's
%    header doesn't get stranded alone at the bottom of a page.
\newlength{\railwidth}
% Set the left column width here. The right column will automatically take the remaining space.
\setlength{\railwidth}{0.22\textwidth}
\newbox\railbox

% \sectionlabel{Title} prints a small standalone section heading
% (e.g. "Employment History"). It does NOT repeat per entry --
% call it once before the first \jobentry/\eduentry/etc. in a section.
\newcommand{\sectionlabel}[1]{%
  \par\vspace{-4pt}%
  \noindent\textcolor{lightgray}{\rule{\textwidth}{0.3pt}}\par\vspace{1pt}%
  \noindent{\bfseries\large #1}\par\vspace{-4pt}%
}

% Generic entry row: left column = metadata (arg #1), right column =
% content (the environment body). A full-width rule sits above the row.
% The metadata is placed in the margin via \makebox, and content flows
% in an indented block that can break across pages.
% Usage:
%   \begin{entryrow}{<left column metadata>}
%     ... right column content ...
%   \end{entryrow}
\newenvironment{entryrow}[1]{%
  \par\needspace{4\baselineskip}\vspace{8pt}%
  \setbox\railbox=\hbox{%
    \begin{minipage}[t]{\railwidth}
      \raggedright #1
    \end{minipage}%
  }%
  \noindent
  \makebox[0pt][l]{%
    \usebox{\railbox}%
  }%
  \begin{adjustwidth}{\dimexpr\railwidth+1em\relax}{0pt}
    \vspace{\dimexpr-\ht\railbox-\dp\railbox\relax}%
    \raggedright
    \setlength{\parskip}{0pt}%
}{%
  \end{adjustwidth}%
  \par
}
